# DashboardM8 - Enterprise Cybersecurity Vulnerability Management Platform

## What is DashboardM8?

DashboardM8 is a powerful, enterprise-grade web dashboard designed for comprehensive vulnerability management and penetration testing operations. Built with modern technologies and security-first principles, it provides security teams with real-time visibility into their attack surface, automated scanning workflows, and actionable intelligence to protect their digital assets.

## Key Features

- **Real-time Vulnerability Tracking**: Monitor discovered vulnerabilities with CVSS scoring, risk ratings, and historical trend analysis
- **Automated Asset Discovery**: Continuous subdomain enumeration, port scanning, and endpoint discovery across your domains
- **Interactive Security Dashboard**: Beautiful, intuitive interface with data visualizations, charts, and customizable reporting
- **Multi-stage Scanning Pipeline**: Orchestrated security scanning workflow from reconnaissance to vulnerability detection
- **Collaborative Workflows**: Built-in chat/support system for team collaboration and issue tracking
- **Smart Notifications**: Real-time alerts via WebSocket for critical vulnerabilities and scan completions
- **Report Generation**: Automated security reports with email delivery and cloud storage integration
- **Scalable Architecture**: Microservices-based design with support for high-volume scanning operations

## Key Components

- **Next.js 15 Frontend**: Modern, responsive web application with server-side rendering
- **Dual Database System**: PostgreSQL for operational data + MongoDB for flexible document storage
- **Real-time Communication**: Socket.IO integration for instant notifications and updates
- **Enterprise Authentication**: NextAuth.js with OAuth support (Google) and JWT security
- **Scanner Integration**: Seamless coordination with 5+ specialized security scanning services
- **Advanced Reporting**: Email integration (SMTP) and cloud storage (AWS S3) for report delivery
- **Comprehensive Logging**: Centralized log management with OpenSearch and Vector
- **Message Queue Architecture**: RabbitMQ for reliable async communication between services

## Technology Highlights

Built with industry-leading technologies: Next.js 15, React 19, TypeScript, Prisma ORM, Tailwind CSS, Tiptap rich text editor, ApexCharts, and a hardened Docker containerized infrastructure.
